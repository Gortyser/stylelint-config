# @mfilin/stylelint-config

Extendable stylelint configuration.

## Installation

```bash
npm i --save-dev @mfilin/stylelint-config
```

## Configuration

In package.json:

```json
"stylelint": {
  "extends": "@mfilin/stylelint-config"
}
```
